# **App Name**: CareerMatch Pro

## Core Features:

- Landing Page: Showcase app overview, features, and signup/sign-in options with a modern UI.
- Multifactor Candidate Job Matching: Score and rank job matches based on skills, experience, location, and extraction confidence with detailed breakdown and explanation.
- AI-Powered Resume Analyzer: Parse resumes, check grammar, provide ATS scores, identify weak points, and offer improvement recommendations as a tool.
- Personalized Career Path Generation: Suggest personalized career paths with course recommendations, skill roadmaps, and upskilling plans as a tool.
- Recruiter Dashboard with Ranked Candidates: Display ranked candidates, parsed resumes, match breakdowns, and suggested interview questions to help recruiters find top talent.
- Job Posting Management: Allow recruiters to post and manage job listings, including necessary information like job title, location, description, and requirements.
- Market Insights Dashboard: Show trending skills, salary range charts, and domain demand graphs to inform candidates and recruiters.
- Authentication and Authorization: Securely authenticate users with email/password, Google OAuth, and LinkedIn OAuth; implement role-based access control for candidates and recruiters.
- Database Persistence: Use PostgreSQL database with tables for users, candidate profiles, recruiter profiles, jobs, applications, skill catalog, and audit logs to store application data, complete with migrations and seed data.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082), a sophisticated and professional hue.
- Background color: Very light gray (#F0F0F0) with only a slight tint of indigo to create a calming neutral surface.
- Accent color: Teal (#008080), which offers a complementary contrast, conveying trustworthiness and growth.
- Body and headline font: 'Inter' sans-serif for a modern and clean reading experience.
- Use a set of professional, consistent icons for job categories, skills, and other key elements.
- Employ a clean and well-structured layout, optimized for both desktop and mobile devices, ensuring intuitive navigation and accessibility.
- Incorporate subtle animations, such as transitions and loading spinners, to enhance user engagement without being distracting.
